Toolbar

Alike the Bootstrap component "Button group" but do not stack horizontally and include an optional group divider.

Unlike the "Button group", you can't define button feature from group container, you need to explicitely do it on each button.

Default divider color is black, use a border color utility class (like border-white) to change it.

Use optional breakpoint classes (break-md, break-lg, etc..) to define when the groups vertically stack.
